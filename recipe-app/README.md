# Recipe App

A beginner-friendly React recipe application built with Vite and JavaScript.

## Features

- React functional components
- `useState` for app state
- Simple recipe browsing
- Add a new recipe name and description
- No TypeScript, no external UI libraries, no backend

## Scripts

- `npm install` — install dependencies
- `npm run dev` — start the local dev server
- `npm run build` — build the app for production
- `npm run preview` — preview the production build
